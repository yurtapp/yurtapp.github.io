# Yurt — Marketing Site

Marketing site for [Yurt](https://yurthome.co), the home management app.

## Installation

```bash
yarn install
```

## Development

Starts a local dev server at http://localhost:3035 with hot reloading.

```bash
yarn run dev
```

## Production Build

Outputs minified JS and CSS to `dist/`.

```bash
yarn run build
```


## Linting & Formatting

```bash
yarn run lint:js
yarn run lint:css
yarn run format
```
